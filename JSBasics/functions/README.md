    <article class="markdown-body entry-content" itemprop="mainContentOfPage"><h1><a id="user-content-functions" class="anchor" href="#functions" aria-hidden="true"><span class="octicon octicon-link"></span></a>Functions</h1>

<h3><a id="user-content-problem-1-english-digit" class="anchor" href="#problem-1-english-digit" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 1. English digit</h3>

<ul>
<li>  Write a function that returns the last digit of given integer as an English word.</li>
</ul>

<p><em>Examples:</em></p>

<table><thead>
<tr>
<th align="center">input</th>
<th align="center">output</th>
</tr>
</thead><tbody>
<tr>
<td align="center">512</td>
<td align="center">two</td>
</tr>
<tr>
<td align="center">1024</td>
<td align="center">four</td>
</tr>
<tr>
<td align="center">12309</td>
<td align="center">nine</td>
</tr>
</tbody></table>

<h3><a id="user-content-problem-2-reverse-number" class="anchor" href="#problem-2-reverse-number" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 2. Reverse number</h3>

<ul>
<li>  Write a function that reverses the digits of given decimal number.</li>
</ul>

<p><em>Example:</em></p>

<table><thead>
<tr>
<th align="center">input</th>
<th align="center">output</th>
</tr>
</thead><tbody>
<tr>
<td align="center">256</td>
<td align="center">652</td>
</tr>
<tr>
<td align="center">123.45</td>
<td align="center">54.321</td>
</tr>
</tbody></table>

<h3><a id="user-content-problem-3-occurrences-of-word" class="anchor" href="#problem-3-occurrences-of-word" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. Occurrences of word</h3>

<ul>
<li>  Write a function that finds all the occurrences of word in a text.</li>
<li>  The search can be case sensitive or case insensitive.</li>
<li>  Use function overloading.</li>
</ul>

<h3><a id="user-content-problem-4-number-of-elements" class="anchor" href="#problem-4-number-of-elements" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Number of elements</h3>

<ul>
<li>  Write a function to count the number of <code>div</code> elements on the web page</li>
</ul>

<h3><a id="user-content-problem-5-appearance-count" class="anchor" href="#problem-5-appearance-count" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 5. Appearance count</h3>

<ul>
<li>  Write a function that counts how many times given number appears in given array.</li>
<li>  Write a test function to check if the function is working correctly.</li>
</ul>

<h3><a id="user-content-problem-6-larger-than-neighbours" class="anchor" href="#problem-6-larger-than-neighbours" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 6. Larger than neighbours</h3>

<ul>
<li>  Write a function that checks if the element at given position in given array of integers is bigger than its two neighbours (when such exist).</li>
</ul>

<h3><a id="user-content-problem-7-first-larger-than-neighbours" class="anchor" href="#problem-7-first-larger-than-neighbours" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 7. First larger than neighbours</h3>

<ul>
<li>  Write a function that returns the index of the first element in array that is larger than its neighbours or <code>-1</code>, if there’s no such element.</li>
<li>  Use the function from the previous exercise.</li>
</ul>
