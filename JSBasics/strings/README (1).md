<h3><a id="user-content-problem-3-sub-string-in-text" class="anchor" href="#problem-3-sub-string-in-text" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 3. Sub-string in text</h3>

<ul>
<li>  Write a JavaScript function that finds how many times a substring is contained in a given text (perform case insensitive search).</li>
</ul>

<p><em>Example:</em></p>

<p>The target sub-string is <code>in</code></p>

<p>The text is as follows:
We are liv<strong>in</strong>g <strong>in</strong> an yellow submar<strong>in</strong>e. We don't have anyth<strong>in</strong>g else. <strong>in</strong>side the submar<strong>in</strong>e is very tight. So we are dr<strong>in</strong>k<strong>in</strong>g all the day. We will move out of it <strong>in</strong> 5 days.</p>

<p>The result is: <code>9</code></p>

<h3><a id="user-content-problem-4-parse-tags" class="anchor" href="#problem-4-parse-tags" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 4. Parse tags</h3>

<ul>
<li><p>You are given a text. Write a function that changes the text in all regions:</p>

<pre><code>&lt;upcase&gt;text&lt;/upcase&gt; to uppercase.
&lt;lowcase&gt;text&lt;/lowcase&gt; to lowercase
&lt;mixcase&gt;text&lt;/mixcase&gt; to mix casing(random)
</code></pre></li>
</ul>

<p><em>Example:</em> We are <code>&lt;mixcase&gt;living&lt;/mixcase&gt;</code> in a <code>&lt;upcase&gt;yellow submarine&lt;/upcase&gt;</code>. We <code>&lt;mixcase&gt;don't&lt;/mixcase&gt;</code> have <code>&lt;lowcase&gt;anything&lt;/lowcase&gt;</code> else.</p>

<p><em>The expected result:</em><br>
We are LiVinG in a YELLOW SUBMARINE. We dOn'T have anything else.</p>

<p><em>Note: Regions can be nested.</em></p>

&lt;/html&gt;

<h3><a id="user-content-problem-11-string-format" class="anchor" href="#problem-11-string-format" aria-hidden="true"><span class="octicon octicon-link"></span></a>Problem 11. String format</h3>

<ul>

<li>  The function should work with up to 30 placeholders and all types.</li>
</ul>

<p><em>Examples:</em></p>

//str = '1, Pesho, 1 text Gosho'
</code></pre>

<li>  Write a function that creates a HTML <code>&lt;ul&gt;</code> using a template for every HTML <code>&lt;li&gt;</code>.</li>

<li>  Replace all placeholders marked with –{…}– with the value of the corresponding property of the object.</li>
</ul>

<p><em>Example:</em> 
<em>HTML:</em> </p>

<pre><code>&lt;div data-type="template" id="list-item"&gt;
 &lt;strong&gt;-{name}-&lt;/strong&gt; &lt;span&gt;-{age}-&lt;/span&gt;
/div&gt;
</code></pre>

<p><em>JavaScript:</em> </p>

<pre><code>var people = [{name: 'Peter', age: 14},…];
var tmpl = document.getElementById('list-item').innerHtml;
var peopleList = generateList(people, template);
//peopleList = '&lt;ul&gt;&lt;li&gt;&lt;strong&gt;Peter&lt;/strong&gt; &lt;span&gt;14&lt;/span&gt;&lt;/li&gt;&lt;li&gt;…&lt;/li&gt;…&lt;/ul&gt;'
</code></pre>
